Import stakeholders

Use this screen to upload stakeholder details from a .csv file.

StakeholdersA person or organisation who has an interest in your projects. are becoming more important to schools as schools become more versatile, working in different ways with different groups. You may have stakeholder data in numerous places e.g. in ParentPay, an Excel spreadsheet on the school business manager’s computer or a number of these. With Arro you can bring all your stakeholder data together in one place and then run campaigns for your projects based on selecting the right stakeholder groups for the campaigns from within Arro.

Importing stakeholder data from a CSV file or from a number of different CSV files enables you to get a lot of stakeholder data into Arro quickly so you can start making this work for you straightaway.

Note: Before you can import stakeholders you need to have a spreadsheet of the information with the extension .csv set up correctly. Click Download Template to set up a new spreadsheet of your data in the correct format.

How do I ...

Create a file to import?
  1. Click Download Template to download Stakeholder-Template.xslx to your PC.
  2. Open this file and save it with a name of your choosing, using Save as type CSV (comma delimited).
  3. Enter the details of the stakeholders you want to import. The information for each stakeholder must be entered for one stakeholder on one line, then the information for the next stakeholder on a new line and so on, until you have entered information for all of the stakeholders.

    4. Note: All entries in the spreadsheet should be in text format and IDs must be prefixed by an apostrophe (‘)

Import data?
  1. Click Choose a file.
  2. Browse to select the .csv file you want to import.
  3. Select the file and click Open.
  4. Click Upload File.

    5. You should see a message saying the file has uploaded successfully. If not, see Correct errors in the import file.

  5. Click Import Stakeholders.
Correct errors in the import file?
  • To remove records from the import, either click Remove alongside an individual record or select a number of records by selecting the check boxes to hte left of the records and then clicking Remove Selected.
  • To abandon the import, click Abandon and confirm at the prompt that you want to abandon the import.
