Record donation

Use this screen to record donations to your project from existing donors or from new donors. When you record donations you can specify if the donor is eligible for the Gift Aid scheme. Gift Aid is a way of achieving a 25% uplift on your donations. To benefit from the Gift Aid scheme, your organisation must be a charity or have charitable status and your donor must be a UK taxpayer. For more information and to ensure that your organisation is eligible to receive gift aid, see the HMRC requirements for claiming gift aid.

How do I ...

Check someone is eligible for Gift Aid?
  • Check the requirements for eligibility on the HMRC website by clicking here.
Find an existing donor?
  • Use the Search by Name box to find an existing donor and then select the check box to the left of the donor name.

    • The search will find and display names which contain the letters you type. If you enter more than one letter, the search finds and displays names which contain the letters consecutively. For example:

    if you enter the display includes
    s Alexander Andrews, Jessica Martin, Marcus Mortimer
    sa James Savage, Lisa Wickham, Sarah Irvine
    san Sandy Bell, Angelica Artisan, Sandra Irvine
Change the gift aid status of an existing donor?
  • You can do this by amending the existing donor details within the Campaigns module.
Add a new donor?
  1. Click +Add Name to open the Create Stakeholder window.
  2. Select the profile type of the donor.
  3. Enter the stakeholder name and address and click Add Contact Details to add phone numbers and addresses.
  4. If the donor is eligible as a gift aid donor:
    • Click Add Gift Aid/Volunteer Status and select the check box, This contact is eligible for Gift Aid.
    • Enter the gift aid donor’s Title, Donor First Name and Donor Last Name.

    Note: It is a legal requirement to retain the information on who is responsible for the gift aid, so if you don’t record the donor’s name here then donations aren’t recorded as donations with gift aid.

  5. If the donor wants to volunteer and help the school, select the check box, This contact would like to volunteer and help my school.
  6. Click Create.
Record a donation?
  1. Enter the Amount of the donation.
  2. Select the donation Type - Cash, Cheque, Direct Credit or ePayment.

    3. Note: ePayment donations are made when donors click the Donations button on the Joinos Web Page for your school or academy.

  3. Select the donation Date.
  4. The donation is given a default DescriptionThis is a voluntary donation. If you want to add your own description, click Add Description and enter the description into the box that appears.
  5. Locate the donor Name.

    6. You can:

    • Add a new donor by clicking + Add Name. Then follow the guidance for Add a new donor.
    • Select an existing donor by selecting the check box to the left of the donor names listed. If you know the donor name or part of the donor name, enter this in the Search by Name box and click the Find icon .
  6. If the donor is eligible for Gift Aid, select Claim Gift Aid on this donation.

    7. If you are selecting an existing donor, and the donor has been set up for Gift Aid, the Gift Aid symbol for the donor will show in colour . If you are unsure whether the donor qualifies for Gift Aid, see the HMRC requirements for claiming gift aid.
