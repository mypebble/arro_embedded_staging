Create new project

Use this screen to create a project from scratch. You might want to do this if you have an unusual project and so cannot find a template on the Choose Template screen to adapt to your own project. Or you might want to do this if you are confident that you know everything you want to say about the project and are happy working with a blank sheet.

How do I create a project?

  1. Enter the Project Title.

    2. Note: We recommend choosing a title which reflects the project aim, is succinct but memorable and will grab attention.

  2. Write a Project Description.

    3. This is where you describe the project as succinctly as you can. It should tell the story of what you are trying to achieve, emphasising the gains the school and the local community will make through the project and should encapsulate the core of the project so that you can use it as an elevator pitch when people in local businesses or other interested parties ask you about your project.

    Note: Click on Hints & Tips for additional help writing your description. To close the window click on Hints & Tips again.

  3. Enter the Start Date.
  4. Enter a Completion Date if you have one. If not, then select No Completion date.
  5. Enter your estimate of how much the project will cost, and so the amount that you need to raise.
  6. Click Create Project.

    7. The project is created and the title and description are displayed in the header section of the View or Edit Project screen.
