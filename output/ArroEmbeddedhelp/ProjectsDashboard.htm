Projects Dashboard

Projects are what drive specific improvements you want to focus on within your school and what it offers as an educational institution and in the wider community. Projects are what you need to plan, generate income for and implement. Use this screen to create and edit projects. New projects are marked as Active by default. Active projects are displayed on the Active tab. You can edit projects and mark them as inactive if required. Projects that you mark as inactive are displayed on the Inactive tab.

Note: You can only edit projects which are active, so if you need to edit an inactive project, mark it as Active first.

How do I ...

Create a project?
  • Click + New Project.

    • You can base your new project on a project template or create from scratch by clicking Create Custom, from the Choose a Project Template screen which appears.

Edit a project?
  • On the Active tab, click the Project Name to display the project details, then click Edit Page to edit the details.
Increase my project strength?

This percentage value indicates how complete your project narrative and description is, so as you fill in the details for the project, the value will increase.

See how much has been raised for a project?

The amount of money raised and the money required for the project is shown in the Funding Target column. Two amounts are shown: the first is the amount of money that has been raised to date, while the second is the total amount of money to raise.

Mark an active project as inactive?
  1. On the Active tab, select the check box before the Project Name.
  2. Click Mark Inactive.
Mark an inactive project as active?
  1. On the Inactive tab, select the check box before the Project Name.
  2. Click Mark Inactive.
