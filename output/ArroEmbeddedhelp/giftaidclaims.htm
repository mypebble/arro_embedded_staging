Gift Aid Claims

Use this screen to manage your Gift Aid claims. You can examine current claims which you need to submit to HMRC, and also look at archived claims you have already made.

There are two steps to submitting a claim:

  1. Creating the claim in Arro: this outputs a specially formatted file (.odsOpen Document Spreadsheet format. A special format required to submit Gift Aid claims to HMRC.) which you need to submit to HMRC.
  2. Submitting the .ods file as a claim to HMRC.

Note: There are limits on the number of lines you can include in a single spreadsheet submitted. November 2014 guidelines state the limit is 1000 lines per file, although you can submit more than one file. This number might change so check the HMRC website for updates to ensure that you comply with the latest guidance.

How do I ...

Know if I can claim Gift Aid?

Check HMRC requirements for claiming gift aid.

Your organisation must have charitable status and the donor must be a UK taxpayer and you should ask them to confirm they are eligible for Gift Aid. You can do this by downloading a declaration form which they can sign, or by asking them to confirm online via email, for example.

Download a Gift Aid declaration form?

You can do this from the Add Stakeholder screen, which you can access from:

  • Campaigns > Stakeholders > +Add New > Manually.
  • Donations > Record Donation > Add Name.
  • Donations > View Donations > +Add New > Add Name.
See the donations I need to claim for?
  1. To preview the outstanding donations, click Preview under Reports.

    2. An on-screen preview of the report is displayed for you to read.

    This shows the number of donors, the potential repayment value, the name and address of the donor, the date the donation was made and the amount donated with the Gift Aid amount.

  2. To download the preview, click Download CSV Preview.

    3. The csvThe name given to a file which stores values that can be read by computer programs. Each line in the file is a record and each record is separated by a comma. file is downloaded.

View archived claims
  1. Click View Report under Reports alongside the claim you want to see the details for.

    2. An on-screen view of the report is displayed for you to read.

    This shows the number of donors, the potential repayment value, the name and address of the donor, the date the donation was made and the amount donated with the Gift Aid amount.

  2. To download the claim information, click Download Claim.

    3. The .odsOpen Document Spreadsheet format. A special format required to submit Gift Aid claims to HMRC. file is downloaded.

Create a claim?
  1. Click Create Claim.
  2. If you have received a statement from HMRC telling you that you have previously claimed too much Gift Aid, enter the excess you have claimed here and it will be offset against the current claim. If you have not received this notification, leave the value as 0.00.
  3. Click Save and Download to download the .ods file you need to submit to HMRC.
Submit the clalim to HMRC?
  • Submit the file to the HMRC, following the HMRC guidelines on how to do this. For more information on submission, see the HMRC guide - Schedule spreadsheet to claim back tax on Gift Aid donations.

    • Note: There are limits on the number of lines you can include in a single spreadsheet submitted. November 2014 guidelines state the limit is 1000 lines per file, although you can submit more than one file. This number might change so check the HMRC website for updates to ensure that you comply with the latest guidance.

    Pay back to HMRC overclaimed Gift Aid?

    If you receive a letter from HMRC saying you have claimed too much gift aid, the next time you submit a claim, enter the amount in the Previously over-claimed Gift Aid box on the Create Claim screen before you click Save & Download.

    The amount you enter is offset against your current claim.
