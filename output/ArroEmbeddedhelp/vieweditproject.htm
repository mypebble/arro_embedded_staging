About the fields
Create Community Page

When you have entered all the details about your project and are ready to begin marketing it, you can create the community page for the project. This is the page which you can use to advertise the project on the Internet and where people can donate time and money to your project.

Project header

The project header contains information that you entered when you created the project - title, description, start and completion date and target. You can also see the project strength. This is an internal measure of how close you are to being ready to market the project for donations through the community page. The higher the percentage, the closer you are. When you first create a project, strength will be around 5%.

Share with Colleagues

You can share the project with colleagues while it is in development. This is different from sharing it through a community page with the wider community. Sharing the project with colleagues is a way for you to invite collaboration on the project from your colleagues. They can provide valuable feedback on the project while it is under development. And you can see the trail of comments on the project Comments tab.

Edit project header

To edit the details in the project header, click Edit.

Note: You can’t edit the Project Strength. This is calculated internally within Arro and is there to help you progress to completing your project.
